# Contributing

Thanks for your interest in this project. Here is how to get started.

## Prerequisites

- Node.js 22+
- npm

## Setup

```bash
git clone git@github.com:sleuthy-sloth/portfolio.git
cd portfolio
npm install
npm run dev      # http://localhost:3000
```

## Before submitting

1. Run `npm run lint` — fix any issues
2. Run `npm run test` — all tests must pass
3. Run `npm run build` — confirm the production build succeeds
4. Keep changes focused. One PR per feature or fix.

## Pull request

- Describe what changed and why
- Reference any related issues
- Screenshots help for visual changes

## Code style

- TypeScript, strict mode
- Tailwind CSS v4 utility classes
- Design tokens via CSS custom properties (no hardcoded hex)
- No em dashes in copy. Use colons, periods, or restructure the sentence.
- Plain language. No filler words.
